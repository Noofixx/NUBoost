# NUBoost
This is a set of files to optimize your PC to the maximum divided into 3 categories: Low, Medium and High(it is recommended to make a restore point before performing any of these configurations)

Low: They are simple configs but they help improve the fps
Medium: They are a little more "complex" configs and can sometimes give errors
High: These are configs designed to optimize the system as much as possible, although in some cases they may cause errors
